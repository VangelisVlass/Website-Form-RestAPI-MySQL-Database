About the Project
The website

In this website you can find two options:
Option 1 - You can add a new entry to your database
Option 2 - You can see, edit and delete your database entries

Option 1

This option redirects you to a simple form. When you fill this Form and hit the Submit button, a new user is stored to the database.

Option 2

This option redirects you to an other webpage, in which you can see the entries stored in the database. Next to each entry you can find two (2) buttons.
The Edit button, pops up a form filled with the attributes of the user. Altering the entries of the form and hitting the Submit button the selected user is updated.
The Delete button, deletes the user and his attributes from the database.
Right after you complete one of the above, the page reloads in order to show the updated data.

The database

The database, named USERS_DB, consists of one and only table named USERS.
Each row of the table stores a USER.
Each USER entry holds seven (7) attributes (Id, Firstname, Lastname, Mobile, Email, Category, Submitted).

Every request from the website to the database, and backwords, is handled by a REST API.

On top of each webpage you can find a Home button which redirects you to the current page.
